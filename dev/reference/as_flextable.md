# Method to transform objects into flextables

This is a convenient function to let users create flextable bindings
from any objects. Users should consult documentation of corresponding
method to understand the details and see what arguments can be used.

## Usage

``` r
as_flextable(x, ...)
```

## Arguments

- x:

  object to be transformed as flextable

- ...:

  arguments for custom methods
