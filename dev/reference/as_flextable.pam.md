# Transform a 'pam' object into a flextable

produce a flextable describing a pam object. The function is only using
package 'broom' that provides the data presented in the resulting
flextable.

## Usage

``` r
# S3 method for class 'pam'
as_flextable(x, digits = 4, ...)
```

## Arguments

- x:

  a [`cluster::pam()`](https://rdrr.io/pkg/cluster/man/pam.html) object

- digits:

  number of digits for the numeric columns

- ...:

  unused argument
