# Format numbers as percentages

Format numeric values as percentages (e.g. `"45.0%"`).

## Usage

``` r
fmt_pct(x)
```

## Arguments

- x:

  numeric values

ft_1 <- mk_par(
  x = ft_1, j = 1, part = "body",
  value = as_paragraph(as_chunk(zz, formatter = fmt_pct))
)
ft_1 <- autofit(ft_1)
