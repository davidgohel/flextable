# Transform a 'summarizor' object into a flextable

`summarizor` object should be transformed into a flextable with method
[`as_flextable()`](https://davidgohel.github.io/flextable/dev/reference/as_flextable.md).

## Usage

``` r
# S3 method for class 'summarizor'
as_flextable(x, ...)
```

## Arguments

- x:

  result from
  [`summarizor()`](https://davidgohel.github.io/flextable/dev/reference/summarizor.md)

- ...:

  arguments for
  [`as_flextable.tabulator()`](https://davidgohel.github.io/flextable/dev/reference/as_flextable.tabulator.md)

## Examples

``` r
z <- summarizor(CO2[-c(1, 4)],
  by = "Treatment",
  overall_label = "Overall"
)
ft_1 <- as_flextable(z, spread_first_col = TRUE)
ft_1 <- prepend_chunks(ft_1,
  i = ~ is.na(variable), j = 1,
  as_chunk("\t")
)
ft_1 <- autofit(ft_1)
ft_1
```
